To learn HTML(Hypertext Markup Language)at first we should know about tags.

So what is so called "tag"?
In HTML (Hypertext Markup Language), tags are the fundamental building blocks used to structure content on a web page. HTML tags consist of keywords enclosed in angle brackets, and they are used to define elements and their properties on a webpage. Here's a basic structure of an HTML tag: html;head;title;body;p;b;u;I;h1...h6;br;A href;img src;ol;ul;li;font;hr;table;table border;caption;th;th;tr;td;sup;sub etc.
